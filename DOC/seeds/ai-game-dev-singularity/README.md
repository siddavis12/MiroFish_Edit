# AI와 게임 개발의 특이점 — 시드 문서 세트

## 개요

AI의 급속한 발전이 게임 개발 방법론과 노동 환경에 미치는 영향을 시뮬레이션하기 위한 시드 문서 세트입니다. 2024~2026년 초까지의 실제 사건, 발언, 데이터를 기반으로 작성되었습니다.

## 시뮬레이션 목적

- 게임 개발 방법론의 AI 시대 변화 관찰
- 게임 회사 경영진과 현장 개발자 간의 인식 격차 분석
- AAA 스튜디오 vs 인디 개발자의 AI 수용 차이 관찰
- 글로벌과 한국 게임 업계의 대응 비교
